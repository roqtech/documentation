# How to add user invites to app

## Get user and tenant data

Before send the invitation, we need to get the user and tenant data. The user data can be retrieved using the `useSession()` method, which returns the user id and user tenanta data from the session.

For example in ROQ BaaS for Next.js:

```ts
import { useSession } from "@roq/nextjs"

const { session } = useSession()

const userId = session?.roqUserId
const tenantId = session?.user?.tenantId
```

## Send invitation

After we get the user and tenant data, we can send the invitation using the [`sendUserInvites()`](/user-invites/api#senduserinvites) method. This method will send an invitation to the user email.

```ts
import { useRoqClient } from '/lib/roq'
 
const roqClient = useRoqClient()
 
const status = async () => {
	const userInvitation = await roqClient.roqPlatform.sendUserInvites({
		userInvites: {
			tenantId: 'ee764883-f2c3-4486-9de6-070345e350af',
			userInvites: [{
				createdByUserId: '530c9a92-71c5-4dce-a6e3-c638c2d0b1bc',
				email: "wick.john@roq.tech",
				locale: "en-US",
				roles: ["content-creator"]
			}]
		}
	})
 
	return userInvitation
}
```
