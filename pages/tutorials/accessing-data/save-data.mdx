# How to save data in Next.js

In this tutorial we will building a form to save data to ROQ using Next.js. We will use the generated SDK to save data to the database.

## Project setup

Before we start, make sure you have completed the following Next.js quickstart:

- [Next.js Quickstart](/next-js)

The quickstart will initialize a Next.js project with ROQ BaaS SDK installed also signup, login, and logout features.

## Save data without relations

To write or update data, we can use `update` method. For instance, to update review data with the review id `f61176b6-ba89-4ec6-8840-1efffb7f51e6`:

```tsx
const updateReviewbyID = async () => {
    const updateResult = await roqClient.review.update({
	    data: { comment: "Highly recommended book" },
		    where: {
			    id: "f61176b6-ba89-4ec6-8840-1efffb7f51e6"
		}
	},)
}
   
updateReviewbyID()
```
The provided review id will be used to update the comment data.
