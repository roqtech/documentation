## ROQ Purpose

ROQ is a managed backend for multi-tenant SaaS applications. It addresses common challenges, allowing you to concentrate on UI and application specific logic.

## Main Features

- Managed Postgres database with instant Rest API
- Visual schema editor
- Organizations (multi-tenancy)
- User roles & permissions
- Authentication with 2FA and social logins
- File handling
- Emails
